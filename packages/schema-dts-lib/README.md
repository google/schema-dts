# schema-dts-lib

Implements root types for JSON-LD vocabularies. This is typically an internal
implementation used by [schema-dts](https://www.npmjs.com/package/schema-dts)
and outputs of [schema-dts-gen](https://www.npmjs.com/package/schema-dts-gen).

Note: This is not an officially supported Google product.
